# HubD

A Jar Engine to deploy complex Website with Java Servlet, MongoDb, Velocity and Maven.

You can see in action in [Agroneo](https://agroneo.com) and more..

#### Functionalities

* Cool HTML Template system
* Json/XML Api
* WebSocket Api
* Content deliver and images manipulation
* OAuth provider
* WebPush
* OAuth client (Gafam)
* Set of tools to Aggregate and Paging easily with MongoDb
* and more..

#### Goodies

* Multiple servers supported
* Seo Ready
* Only Mongodb cache
* LightWeight and performance oriented
* and more..
