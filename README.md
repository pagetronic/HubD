# HubD

A Jar Engine to deploy complex Website

* Performance: Java Servlet
* NoSQL: MongoDb
* HTML Engine: Velocity
* Management: Maven

You can see in action on [Agroneo](https://agroneo.com) and [GitHub example](https://github.com/pagetronic/Agroneo).

#### Functionalities

* Cool [HTML Template](hubd/html) system
* Complex && Simplified [Servlets](hubd/src/live/page/web/system/servlet)
* [Json/XML](hubd/src/live/page/web/system/json) [Api](hubd/libs/api.js)
* [Love Ajax](hubd/libs/ajax.js)
* [Live](hubd/src/live/page/web/system/socket/SocketDb.java) [WebSocket](hubd/src/live/page/web/system/socket) [Api](hubd/libs/socket.js)
* [Images/Docs manipulation](hubd/src/live/page/web/blobs)
* [OAuth provider](hubd/src/live/page/web/api)
* [WebPush](crond/src/live/page/notice)
* [OAuth client](hubd/src/live/page/web/system/sessions/oauth)
* Set of tools to [Aggregate and Paging](hubd/src/live/page/web/system/db) easily with MongoDb
* [Language processor](hubd/src/live/page/web/system/Language.java)
* [and more](hubd/src/live/page/web/system/Settings.java)..

#### Goodies

* Multiple servers supported
* Seo Ready
* Only Mongodb cache
* LightWeight and performance oriented
* and more..
