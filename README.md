# HubD
A Jar Engine to deploy complex Website with Java Servlet, MongoDb and Velocity

You can see in action on [Agroneo](https://agroneo.com).

## Functionalities

* Cool HTML Template system
* Json/XML Api
* WebSocket Api
* Content deliver and images manipulation
* OAuth provider
* WebPush
* OAuth client (Gafam)
* Set of tools to Aggregate and Paging easily with MongoDb
* and more..

## Goodies

* Multiple servers supported
* Seo Ready
* Only Mongodb cache
* LightWeight and performance oriented
* and more..
